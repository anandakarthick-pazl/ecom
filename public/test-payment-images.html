Directory Check

Common storage paths that should exist:

  • Physical path: D:\source_code\ecom\storage\app\public\payment-methods\
  • Symlink path: D:\source_code\ecom\public\storage\payment-methods\
  • URL path: http://greenvalleyherbs.local:8000/storage/payment-methods/

Troubleshooting:

  1. Check if the storage symlink exists: php artisan storage:link
  2. Verify image files exist in storage/app/public/payment-methods/
  3. Check database records have correct image paths
  4. Test direct URL access in browser
